# När kommer glassbilen?

Kolla när glassbilen kommer, i din terminal.

0. pnpm install
1. Skriv in din latitud och longitud i filen .env (med `lat=nn.nnnnnn` och `long=nn.nnnnn` på var sin rad).
2. Kör node main.js
